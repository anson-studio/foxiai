# 🦊 FoxiAI

商业视觉创作平台 - AI 修图、生成、视频

## 功能

- 产品精修 / 场景精修 / 人像修图
- 换装试衣 / 模特换姿 / 模特替换
- 文生图 / 图生图 / 文生视频 / 图生视频
- 无限画布 - 节点式工作流
- 智能 AI 图像分析 (GPT-4o Vision)

## 技术栈

- 前端: 纯 HTML + JavaScript (无框架)
- 后端: Vercel Serverless Functions
- API: ofox.ai (统一AI模型聚合平台)

## 部署

一键部署到 Vercel:

```
git push origin main
```

Vercel 会自动检测并部署。
